Lógica de Programação com Python

Objetivo do Curso

Neste curso de Lógica de Programação o(a) estudante será introduzido(a) ao mundo da Programação. Durante o curso, serão apresentados conceitos como: variáveis, funções de entrada e saída, comandos de controle de fluxo (seleção e repetição), assim como os conceitos de funções, arrays, dicionários e estruturas. O objetivo final de aprendizagem é que o(a) estudante consiga de forma objetiva distinguir quando e onde utilizar cada estrutura e/ou comando aprendido na resolução de problemas reais de desenvolvimento

Aulas Programadas

Módulo 1: Introdução a Programação e Controle de Fluxo

  1. Introdução a Algoritmos.
  2. Introdução a Variáveis, Tipo de Dados e Entrada e Saída.
  3. Introdução a Controle de Fluxo: Operadores Relacionais, Comparações, if ... else
  4. Introdução aos Operadores Lógicos e if ... elif ... else, Escopo de variáveis
  5. Introdução a Repetição: while, contador e sentinela
  6. Reptição: Exercícios: contador, somatório e média
  7. Repetição: for, contador, função range, somatório, média, maior, menor
  8. Repetição: Exercícios: Teste de Mesa
  9. Repetição: while ... else, Validação de Entrada do Usuário.
  10. Repetição: Exercícios: Teste de Mesa e Validação de Formulários.

Módulo 2: Modularização de Sistemas

  1. Introdução a Módulos e Funções: a funções: input, print, definição de funções
  2. Introdução aos parâmetros, argumentos e retorno.
  3. Introdução ao escopo, tempo de vida de variáveis
  4. Exercícios de Funções: Como dividir as responsabilidades em funções
  5. Introdução a pilha de chamada de função: passagem por valor
  6. Introdução ao retorno de múltiplos valores
  7. Introdução a recursão: Identificar o Caso Base e o Passo Recursivo
  8. Recursão vs Iteração
  9. Exercícios de Funções: Como modularizar um sistema

Módulo 3: Listas: vetores, matrizes

  1. Introdução a Listas: criação ([], list()), inserir novos elementos (função append), índices
  2. Listas: lendo vários valores em um loop for, modificação de valores
  3. Funções: len, append, insert, pop, remove
  4. Funções: clear, sort, reverse, count
  5. Procurando um valor: busca linear, operador in
  6. Manipulação de índices: intervalo, pegando os últimos índices, pegando os primeiros índices, função: extend()
  7. Estudo de caso: Um pequeno caso de cadastro de reservas
  8. Introdução a bidimensional arrays: acessando row, column; resolvendo problemas multidimensionais
  9. Linearizando uma matriz
  10. Exercícios 1: Implementando a nossa biblioteca de Strings: toLowerCase, toUpperCase, find
  11. Exercícios 2: Implementando a nossa biblioteca de Strings: contains, startsWith, endsWith
  12. Exercícios 3: Implementando a nossa biblioteca de Strings: replaceChar, replaceSubString, equals, equalsIgnoreCase

Módulo 4: Tuplas

  1. Introdução a Tuplas: criação: ( (), tuples() )
  2. Tuplas: acesso aos itens, intervalos, últimos índices
  3. Modificando as tuplas: remover, adicionar
  4. Descompactando uma tupla, iterações em loops, juntando tuplas
  5. Funções: count(), index()
  6. Exercícios: Exemplos de utilização prática das tuplas

Módulo 5: Dicionários

  1. Introdução aos Dicionários: criação, definição de campos. Acessando os dados: ["key"], get("key")
  2. Acessando Dicionários: ["key"], get("key"); keys(), values(), items()
  3. Verificando se a chave existe (operador in), adicionando nova chave
  4. Alterando os valores, função: update(), pop(), popitem(), del, clear()
  5. Iterando sobre dicionários: loop for
  6. Estudo de Caso: Cadastro de clientes numa rede de hotelaria
  7. Estudo de Caso: Cadastro de quartos numa rede de hotelaria
  8. Estudo de Caso: Cadastro de reservas numa rede de hotelaria
